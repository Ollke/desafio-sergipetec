# 📦 Desafio Técnico SergipeTec

Aplicação desenvolvida em **Spring Boot** para controle de usuario, produtos e pedidos.

🛠️ Pré-requisitos
Antes de começar, você vai precisar ter instalado em sua máquina:

* Git
* Docker
* Docker Compose

## ✅ Guia de Configuração do Ambiente
Para subir o ambiente de desenvolvimento, siga os passos abaixo:

* docker-compose up -d

## Acesse a aplicação:
* Após o Docker finalizar a inicialização, a aplicação estará disponível em: http://localhost:8080/.
